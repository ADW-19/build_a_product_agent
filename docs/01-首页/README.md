# build_a_product_agent: 构建生产级 AI Agent

> 本项目是一份 AI Agent 后端开发技术手册，系统讲解如何从零设计并实现生产级 Agent 系统。

 [快速开始](#快速开始) · [目录结构](#目录结构) · [技术栈](#技术栈) · [English](./README-en.md)

---

## 项目简介

> 作者：Andy Yanqi Wang (ADW-19) · 中国上海
>
> 官方社区：
>
> * 小红书ID：ADW_AI

这是一个**纯文档项目**，专注于 AI Agent 后端开发的核心知识点。每个章节都是一篇独立的技术文章，涵盖：

- 技术选型与架构设计
- 关键模块开发指南
- Agent 通路与多 Agent 协作
- 系统测试与上线验证

> 目标读者：具备一定编程基础的开发者，以"工业界标准 → 为什么课堂不教 → 你应该怎么写"为叙事线。

## 快速开始

```bash
# 克隆项目
git clone https://github.com/ADW-19/build_a_product_agent.git
cd build_a_product_agent

# 阅读顺序建议
# 1. 第1章：技术选型（了解整体技术栈）
# 2. 第2章：开发基本要求（编码规范）
# 3. 第3章：模块开发（深入实现细节）
# 4. 第4章：Agent通路（单Agent与多Agent协作）
# 5. 第5章：系统测试（上线前的测试体系）
```

## 技术栈

> 本手册围绕以下技术栈展开

| 组件               | 选型       | 版本要求 |
| ------------------ | ---------- | -------- |
| 语言               | Python     | 3.11+    |
| Web 框架           | FastAPI    | ≥ 0.115 |
| Agent 编排         | LangGraph  | ≥ 0.3   |
| LLM 工具层         | LangChain  | ≥ 1.0   |
| 数据校验           | Pydantic   | v2       |
| 业务数据库         | PostgreSQL | -        |
| 缓存/会话/限流     | Redis      | -        |
| 向量库（长期记忆） | Milvus     | -        |
| 消息队列           | RabbitMQ   | -        |
